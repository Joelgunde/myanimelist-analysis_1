💡 Key Insights

📺 Content Strategy

12-24 episode series have the highest average ratings. Quality over quantity matters!

🎬 Type Performance

Movies consistently outperform TV shows with an average rating of 7.8/10.

🎭 Genre Trends

Action dominates (35%), but Drama is rising. Genre mix is crucial for success.

📈 Production Peak

Anime production peaked 2015-2020. Clear market saturation trend observed.

⭐ Rating Patterns

Bell-curve distribution centered at 7.2/10. Most anime rated between 6.5-8.0.

🎯 Optimal Format

The sweet spot for anime success is 1-2 seasons with consistent quality.

🚀 Getting Started

Want to run this analysis yourself? Follow these steps:

Prerequisites

  • Python 3.8 or higher
  • pip package manager
  • MyAnimeList dataset from Kaggle

Installation

git clone https://github.com/Joelgunde/myanimelist-analysis.git
cd myanimelist-analysis
pip install -r requirements.txt
python src/analyze.py
python src/visualizations.py

🔬 Methodology

1. Data Collection

Downloaded MyAnimeList dataset from Kaggle containing 24,905 anime titles with 24 features.

2. Data Cleaning

Handled missing values, removed duplicates, converted data types, and treated outliers.

3. Feature Engineering

Created derived features including year extraction, rating categories, and popularity tiers.

4. EDA

Performed comprehensive analysis from univariate to multivariate patterns.

5. Visualization

Built interactive dashboard with 8+ Plotly charts for data exploration.

6. Insights

Documented key findings and actionable recommendations for stakeholders.
